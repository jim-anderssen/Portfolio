## Projects in This Repo

- **Server management scripts** — Python command-line utilities for managing a shared server with tens of thousands of files. Includes tools for finding old or duplicate files via hash algorithms, detecting deeply nested paths, building directory tree graphs, and bulk-compressing images.

- **Indoor air analysis project** — Analysis of indoor climate sensor data from a factory experiencing mould issues. Covers the full pipeline from data collection and quality assessment through cleaning, exploratory analysis, and visualisation across 15+ sensors.

- **Cyanobacteria satellite observation model** — Remote sensing analysis of cyanobacteria using satellite data, including time series generation, coordinate handling, and supporting notebooks.

- **Older work** — University projects including wind and solar power assessments.
